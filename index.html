Project 1: Physics Simulation in Python

This project simulates the motion of an object under the influence of gravity using Python. It calculates position and velocity over time.

View Code
# Python code for physics simulation import numpy as np import matplotlib.pyplot as plt # Constants g = 9.81 # acceleration due to gravity (m/s^2) initial_velocity = 0 # initial velocity (m/s) time = np.linspace(0, 10, 100) # time array (seconds) # Calculating position and velocity position = initial_velocity * time + 0.5 * g * time**2 velocity = initial_velocity + g * time # Plotting position and velocity plt.subplot(2, 1, 1) plt.plot(time, position) plt.title('Position vs Time') plt.xlabel('Time (s)') plt.ylabel('Position (m)') plt.subplot(2, 1, 2) plt.plot(time, velocity) plt.title('Velocity vs Time') plt.xlabel('Time (s)') plt.ylabel('Velocity (m/s)') plt.tight_layout() plt.show()

Project 3: Numerical Integration

This project demonstrates numerical integration using the trapezoidal rule to estimate the area under a curve.

View Code
# Python code for numerical integration (trapezoidal rule) import numpy as np # Function to integrate def f(x): return x**2 # Trapezoidal rule function def trapezoidal_rule(a, b, n): x = np.linspace(a, b, n) y = f(x) return np.trapz(y, x) # Estimate the integral of f(x) = x^2 from 0 to 1 integral_value = trapezoidal_rule(0, 1, 1000) print(f"Estimated integral value: {integral_value}")
